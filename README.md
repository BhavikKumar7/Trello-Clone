# 🧩 Trello Clone -- Full Stack Project

A full-stack Trello-like project management application.

## 🚀 Tech Stack

-   React, Node.js, Express
-   MySQL (Aiven)
-   JWT, bcrypt

## ⚙️ Setup

### Backend

npm install nodemon server.js

### Frontend

npm install npm run dev

## 📁 Structure

client/ server/

## 🗄️ Database

users, boards, lists, cards, labels, card_labels, card_members,
board_members, checklist_items

## 🚀 Deployment

Frontend: Vercel Backend: Render DB: Aiven
